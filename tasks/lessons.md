# Lessons

## 2026-07-02: サブエージェント稼働中の作業ツリー操作禁止

- **事象**: ベースライン確認のため `git stash` / `git stash pop` を実行したが、その時点で
  バックグラウンドのサブエージェントが同じ作業ツリーにファイルを書き込み中だった。
  stash がサブエージェントの変更(main.py)まで巻き込み、pop 時の競合や、
  一時的に巻き戻った状態でサブエージェントがテストを実行するリスクがあった。
- **ルール**: サブエージェントが作業ツリーで稼働中は、stash / checkout / reset 等の
  ツリー全体を変異させる git 操作をしない。ベースライン検証が必要なら
  `git worktree` で別ディレクトリに切り出すか、サブエージェント完了後に行う。
